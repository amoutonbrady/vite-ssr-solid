# _Vite Solid_ SSR

A SSR implementation using [Solid.js](https://solidjs.com/) & [Vite](https://vitejs.dev/)

## Usage

First get the Repository

```bash
npx degit amoutonbrady/vite-ssr-solid <app-dir>
```

then cd into your app dir

```bash
cd <app-dir>
```

and install the dependencies

```bash
pnpm i
```
